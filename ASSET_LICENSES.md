# Asset licenses

All PNG files under `public/assets/generated/` are original procedural artwork created by
`tools/generate-assets/generate.ts` for this project.

| Files                           | Author                            | Source URL | License | Modifications                                                              | Download date |
| ------------------------------- | --------------------------------- | ---------- | ------- | -------------------------------------------------------------------------- | ------------- |
| `public/assets/generated/*.png` | Original work for this repository | N/A        | CC0-1.0 | Generated from the Level 0 palette and material recipes in this repository | N/A           |
| Runtime audio from `src/audio/` | Original work for this repository | N/A        | CC0-1.0 | Synthesized from deterministic signal recipes with the Web Audio API       | N/A           |

The deterministic `manifest.json` records the generator version, per-asset seed, dimensions, role,
SHA-256 digest and SPDX license identifier. No image, texture, model or sound was downloaded to
create the game assets.
