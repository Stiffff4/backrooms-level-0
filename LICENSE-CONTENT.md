## Elementos adaptados

El juego adapta expresamente:

- el nombre **Level 0**;
- la descripción de un espacio de oficinas amarillo, húmedo, fluorescente y aparentemente infinito;
- los elementos arquitectónicos generales descritos por la fuente;
- el zumbido fluorescente como parte central del ambiente;
- la idea de una pared parpadeante como salida hacia Level 1.

## Modificaciones realizadas

La adaptación transforma esos elementos en una experiencia first-person interactiva y breve para
navegador. Incorpora un grafo procedural reproducible por semilla, streaming con memoria acotada,
salas modulares, dirección visual pixelada, síntesis de audio Web Audio, anomalías ambientales sin
entidades, azar controlado para la salida y una transición final original.

No se copiaron textos extensos ni imágenes de la página fuente. Las texturas, geometría, interfaz y
recetas de audio se crearon específicamente para este repositorio.
